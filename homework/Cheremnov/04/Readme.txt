Сборка программы производится с использованием утилиты make. (Makefile для этого находится в папке).
Работа программы:
Необходимо ввести число:
0 для тестирования операций присваивания элемента
1 для тестирования ошибки выхода за границу в случае строк
2 для тестирования ошибки выхода за границу в случае столбцов
3 для тестирования ошибки несовпадения размерности вектора и матрицы
4 для тестирования сравнений на равенство и неравенство матриц
5 для тестирования операции умножения матрицы на число и на вектор
